# Project Rules

## 프로젝트 개요
웹/앱 반응형 게임. GitHub Pages(github.io)로 배포하는 정적 웹 앱.

---

## 프로젝트 목표

**학습 RPG — 타워 기반 지식 탐험 게임**

- 플레이어가 학습하고 싶은 주제를 **RPG 게임 형식**으로 경험할 수 있도록 구현
- 주제(컨텐츠)별로 **타워(Tower)** 가 존재하며, 각 타워는 독립된 학습 세계를 표현
- 타워 내부는 층(Floor) 또는 스테이지 구조로 나뉘어 난이도가 점진적으로 상승
- 학습 요소(퀴즈, 개념 설명, 문제 풀이 등)를 전투/탐험 메카닉에 녹여 설계

### 컨텐츠 구조 원칙
- **타워 = 학습 주제** (예: 수학 타워, 영어 타워, 알고리즘 타워 등)
- 각 타워는 독립적으로 추가/확장 가능한 모듈 구조로 개발
- 타워 간 공통 RPG 요소(캐릭터, 경험치, 인벤토리 등)는 공유
- 새 타워 추가 시 기존 코드를 최소한으로 수정하도록 설계 (플러그인 방식 지향)

### 스텟(Stat) 시스템
스텟은 두 축으로 구성된다:

**1. RPG 성장 스텟** — 캐릭터 육성
- 레벨업, 전투, 아이템 등으로 올라가는 고전적인 RPG 스텟
- 예: 체력, 공격력, 방어력, 민첩, 마나 등
- 타워 공략 난이도와 직결

**2. 학습 성취 스텟** — 실력 반영
- 퀴즈 정답률, 학습 완료도, 반복 횟수 등 학습 행동에서 자동 산출
- 예: 이해도, 암기력, 응용력 등 주제별 수치
- 학습 스텟이 높을수록 해당 타워의 RPG 스텟 보너스 제공 (두 축 연동)

**연동 원칙**
- 학습 없이 RPG 스텟만 올리는 것은 불가 또는 제한 — 학습이 성장의 핵심 경로
- 스텟 데이터는 영구 저장 대상 (IndexedDB 또는 클라우드)

## 반응형 규칙

- 기준 해상도를 하나 정하고 (예: 1280×720 또는 360×640) 비율 기반으로 scale
- CSS `vw/vh`, `clamp()`, `aspect-ratio` 활용 — 고정 px 금지
- 터치(mobile)와 키보드/마우스(desktop) 입력을 동시에 지원
- 브레이크포인트: `480px` (모바일), `768px` (태블릿), `1024px+` (데스크탑)
- 게임 캔버스는 항상 화면 중앙 배치, 여백은 letterbox 처리
- 세로/가로 모드 전환 대응 (`orientationchange` 이벤트 처리)

---

## GitHub Pages 배포 규칙

- 백엔드 서버 없음 — 모든 로직은 클라이언트 사이드
- 빌드 결과물은 `docs/` 또는 별도 `gh-pages` 브랜치에 배포
- 빌드 도구: **Vite** 사용 (`base` 옵션을 레포 이름에 맞게 설정)
- 외부 API 호출 시 CORS 이슈 주의 — HTTPS 전용
- 에셋(이미지, 오디오)은 상대 경로 사용, 절대 경로 금지

---

## 영구 데이터 저장 규칙

### 우선순위
1. **localStorage** — 간단한 설정, 최고 기록 등 소규모 데이터
2. **IndexedDB** — 세이브 데이터, 인벤토리 등 구조화된 대용량 데이터
3. **외부 서비스** (선택) — Firebase / Supabase 로 클라우드 저장소 연동 가능

### 규칙
- 저장 키는 네임스페이스 접두사 사용: `game::{key}`
- 저장 전 항상 JSON 직렬화, 읽을 때 파싱 실패 대비 try/catch
- 저장 데이터에는 반드시 `version` 필드 포함 (마이그레이션 대비)
- 민감 정보(비밀번호 등) localStorage 저장 금지
- 저장 실패 시 사용자에게 알림 표시

## 기술 스택 (기본값)

- **언어**: TypeScript
- **게임 엔진**: Phaser.js (변경 시 이 파일 업데이트)
- **빌드**: Vite
- **스타일**: CSS Modules 또는 순수 CSS

## 코드 규칙

- 탭 대신 스페이스 2칸
- 파일명은 `kebab-case`, 클래스명은 `PascalCase`, 변수/함수는 `camelCase`
- 게임 씬/상태는 각각 별도 파일로 분리
- 매직 넘버 금지 — 상수 파일(`constants.ts`)에 정의
